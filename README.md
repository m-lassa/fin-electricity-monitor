# ENTSO-E Electricity Prices for Finland

A web application that fetches electricity prices for the current day from the ENTSO-E API and displays them on a dynamic chart using [Chart.js](https://www.chartjs.org/). Built with [Node.js](https://nodejs.org/), [Express](https://expressjs.com/), [EJS](https://ejs.co/), and CSS.


## Features
- Fetches real-time electricity prices for Finland from the ENTSO-E API.
- Displays the prices in an interactive and responsive chart powered by Chart.js.
- Automatically updates the chart with the current day's data.
- Simple, clean UI for easy viewing of electricity price trends.

## Technologies
- **Backend:** Node.js, Express.js
- **Frontend:** EJS, CSS, Chart.js
- **Data Source:** [ENTSO-E Transparency Platform](https://transparency.entsoe.eu/)
